Goal

My goal is to display a personal project to close friends and colleagues to see. Therefore, my design needs to be professional and clean, while retaining the colorful and fun tone of the game.

Notes

I found it challenging to work with CSS grid however, I believe I've learned more about CSS grid through this project. Additionally, I changed the content that was on the original outline design.
