Project 2 Interface Sketch

Goal

My plan for this project is to create an amiibo finder with the amiibo API. It would be used by fans and collectors of amiibo to search for amiibos by game series, amiibo series, and name. This would give fans and collectors of amiibo an organized way to search for amiibos.

Notes

At first, I found it challenging to learn how to navigate the amiibo API, however, the amiibo API documentation made it easier to understand. I based my page layout on Nintendo's official amiibo webpage. I found the minimalistic approach appropriate since the amiibo figures pop out more against a plain background.
